# LinkedIn Profile Reviewer Pro

A Python-based tool that analyzes a user's LinkedIn profile headline and About section to provide feedback and improvement suggestions.

## Features

* Reviews LinkedIn headline quality
* Detects AI and Machine Learning related keywords
* Checks for GitHub portfolio mention
* Verifies project experience mention
* Identifies career objective statements
* Generates a profile score out of 100
* Provides personalized improvement suggestions

## Technologies Used

* Python
* Conditional Statements
* Loops
* String Processing
* Lists

## Learning Outcomes

This project helped me understand text analysis, user input handling, conditional logic, and building rule-based recommendation systems.
